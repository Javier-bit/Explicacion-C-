Mi codigo

Para poder comenzar,se debera colocar el include y posteriormente se abriria el main()

Despues dentro del main,se comenzara por colocar un printf

Y dentro del printf se colocaria algo como "inserta una cadena de caracteres"

Despues de eso se pondra un scanf que guardararia la cadena de caracteres en una variable que le podriamos llamar oracion

Cuando se tenga la variable guardada,se usara el if para comparar que sea un string

Si es asi le dejaremos continuar y con una funcion que contara el numero de letras que tiene la cadena de caracteres ingresada

Y por ultimo le imprimiremos el numero de la cadena de caracteres

De lo contrario se usara un else el cual le imprimira que esa no es una cadena de caracteres valida

Esto se repetira 3 veces mas, una ves se gaste sus 3 oportunidades se cerrara el programa
